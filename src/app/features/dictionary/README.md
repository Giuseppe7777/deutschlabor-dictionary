# Dictionary Feature

This folder contains the dictionary search feature.

Use this folder for:
- dictionary search page
- search input UI
- autocomplete UI
- language pair selector
- search results UI
- searched words UI
- dictionary-specific services
- dictionary-specific models

Do not use this folder for:
- global authentication state
- global HTTP interceptors
- global route guards
- account settings UI
- training session UI
- reusable shared UI components used by many features

Rule:
Dictionary feature code belongs here when it is directly related to word search, language pairs, search results, or searched words.
