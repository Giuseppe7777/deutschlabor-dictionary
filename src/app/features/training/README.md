# Training Feature

This folder contains the word training feature.

Use this folder for:
- training overview page
- words due for review UI
- review list UI
- training start UI
- training session UI
- answer input UI
- training result UI
- training-specific services
- training-specific models

Do not use this folder for:
- global authentication state
- global HTTP interceptors
- global route guards
- dictionary search UI
- account settings UI
- reusable shared UI components used by many features

Rule:
Training feature code belongs here when it is directly related to reviewing, practicing, grading, or repeating searched words.
